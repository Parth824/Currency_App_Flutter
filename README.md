# *Currency App*
Currency app made with Flutter in India.

---

</div>


## 🗒 About

These are Currency App in which you can Search your needs by Currency.

## ⚙️ Features

    - Search Features
    - Category-wise Currency display
    - MVC directory structure
    - switch betweeen light or dark theme.
